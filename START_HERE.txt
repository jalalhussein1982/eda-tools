═══════════════════════════════════════════════════════════════
  🎯 WHAT IT DOES
═══════════════════════════════════════════════════════════════

STAGE 1: Upload Data
  • CSV or Excel files up to 500MB
  • Automatic data type detection

STAGE 2: Variable Selection
  • Explicit IV/DV designation
  • Drag-and-drop interface

STAGE 3: Data Quality Assessment
  • Missing value detection + handling strategies
  • Outlier detection with IQR method
  • User decides on each issue

STAGE 4: Distribution Analysis
  • Shapiro-Wilk normality tests
  • Anderson-Darling tests
  • Skewness & kurtosis
  • Histogram visualizations

STAGE 5: Configure Analysis
  • Correlation methods (Pearson, Spearman, Kendall)
  • VIF for multicollinearity
  • Linear/polynomial regression
  • Assumption tests (homoscedasticity, independence)
  • Significance level selection

STAGE 6: Analysis & Results
  • Runs all analyses (~30-60 seconds)
  • Generates interactive HTML report
  • Plain-language interpretations
  • Confidence scoring for findings
  • Downloadable report

FEATURES:
  ✓ Multiple correlation methods
  ✓ VIF multicollinearity detection
  ✓ Regression diagnostics
  ✓ Assumption testing
  ✓ Q-Q plots, histograms, scatter plots
  ✓ Confidence intervals
  ✓ APA-style output
  ✓ Configuration save/load

═══════════════════════════════════════════════════════════════
  📊 TECHNICAL SPECIFICATIONS
═══════════════════════════════════════════════════════════════

ARCHITECTURE:
  • Client-side: HTML5 + CSS3 + JavaScript ES6+
  • Python runtime: Pyodide (WebAssembly)
  • Libraries: NumPy, Pandas, SciPy, Statsmodels, scikit-learn
  • Visualization: Plotly.js
  • Offline: Service Worker API

BROWSER REQUIREMENTS:
  • Chrome 90+, Firefox 89+, Safari 15.2+, Edge 90+
  • JavaScript enabled
  • 4GB+ RAM recommended

PERFORMANCE:
  • First load: 30-60 seconds (downloads Python runtime)
  • Subsequent loads: 3-5 seconds (cached)
  • Analysis time: 30-60 seconds (typical dataset)
  • Max dataset: 500MB (browser memory limit)
