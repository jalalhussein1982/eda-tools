QUICK START:
1. cd eda-tool
2. python -m http.server 8000
3. Open http://localhost:8000
4. Wait 30-60 seconds for Python to load
5. Click "Health Survey" sample dataset
6. Follow the wizard (5 minutes total)

DEPLOYMENT:
- Upload to any web server (Apache, Nginx, GitHub Pages)
- Requires HTTPS for offline capability
- No backend/database needed
- Zero maintenance required
- See DEPLOYMENT.md for details
